Prototipo 3D

Prototipo principal

Se presenta un modelo tridimensional del campus, incluyendo pasillos, salones y áreas interactivas. Permite al usuario recorrer el campus de manera virtual, observando detalles de cada espacio y cómo interactuar con ellos.

Características del prototipo

  • Navegación intuitiva: el usuario puede moverse fácilmente por el campus.
  • Interacción con espacios: permite seleccionar aulas y laboratorios para obtener información.
  • Modelo realista: se utilizan texturas y animaciones para aumentar el realismo.
